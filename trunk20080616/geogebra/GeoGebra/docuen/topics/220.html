Dilate

Dilate[point A, number f, point S]: Dilates point A from point S using factor f

Dilate[line h, number f, point S]: Dilates line h from point S using factor f

Dilate[conic c, number f, point S]: Dilates conic section c from point S using factor f

Dilate[polygon poly, number f, point S]: Dilates polygon poly from point S using factor f. Note: New vertices and segments are created too.

Dilate[image pic, number f, point S]: Dilates image pic from point S using factor f

 

Note: Also see mode  Dilate object from point by vector
