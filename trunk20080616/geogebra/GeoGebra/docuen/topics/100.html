4.1.1.        Change Values

Free objects can be changed directly whereas dependant objects can’t. To manipulate a free object’s value overwrite it by entering the new value in the input field (see Direct Input).

Example: If you want to change the value of an existing number a = 3, type a = 5 into the input field and press the Enter key.

 

Note: Alternatively, this can be done in the algebra window by choosing Redefine in the Context Menu or by double clicking an object in  Move mode in the algebra window.
