3.1.9.        Properties Dialog

The properties dialog allows you to modify properties of objects (e.g., color, line style). You can open this dialog either by right clicking (MacOS: Apple + click) an object and selecting Properties, or by selecting Properties in the Edit menu.

 

In the properties dialog objects are organized by types (e.g., points, lines, circles) which makes it easy to handle large numbers of objects. You can modify the properties of selected objects within the tabs on the right side. Close the properties dialog when you are done with changing properties of objects.
