PNG – Portable Network Graphics

This is a pixel graphics format. The higher the resolution (dpi), the better the quality (300dpi will usually suffice). PNG graphics should not be scaled subsequently to avoid a loss of quality.

 

PNG graphic files are well suited for the use on web pages (html) and with Microsoft Word.

Note: Whenever you insert a PNG graphic file into a Word document (menu Insert, Image from file) make sure that the size is set to 100 %. Otherwise the given scale (in cm) would be changed.
