4.3.19.   Geometric Transformations

If you assign one of the following commands to a new name, a copy of the moved object will be produced.

Note: The command Mirror[A, g] mirrors point A at line g and changes point A's location. Entering B = Mirror[A, g] will produce a new point B while point A is left unchanged.
