Integral

Integral[function f, number a, number b]: Definite integral of function f(x) from number a to b. Note: This command also draws the area between the function graph of f and the x-axis.

Integral[function f, function g, number a, number b]: Definite integral of the difference of the functions f(x) - g(x) from number a to number b. Note: This command also draws the area between the function graphs of f and g.

 

Note: See Indefinite integral
