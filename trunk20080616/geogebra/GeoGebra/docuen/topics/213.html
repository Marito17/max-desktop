Sequence

Sequence[expression e, variable i, number a, number b]: List of objects created using expression e and index i that ranges from number a to number b.
Example: L = Sequence[(2, i), i, 1, 5] creates a list of points whose y-coordinates range from 1 to 5

Sequence[expression e, variable i, number a, number b, number s]: List of objects created using expression e and index i that ranges from number a to number b with given step size s.
Example: L = Sequence[(2, i), i, 1, 5, 0.5] creates a list of points whose y-coordinates range from 1 to 5 with a step size of 0.5.

 

Note: Since the parameters a and b are dynamic you could use slider variables there.
