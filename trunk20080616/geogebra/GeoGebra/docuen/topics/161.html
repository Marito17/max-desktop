PerpendicularVector

PerpendicularVector[line g]: Perpendicular vector of a line g. Note: A line with equation ax + by = c has the perpendicular vector (a, b).

PerpendicularVector[vector v]: Perpendicular vector of a vector v. Note: A vector with coordinates (a, b) has the perpendicular vector (- b, a).
