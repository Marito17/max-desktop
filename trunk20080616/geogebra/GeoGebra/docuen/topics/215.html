Iteration

IterationList[function f, number x0, number n]: List L of length n+1 whose elements are iterations of        function f starting with the value x0.      
Example: After defining function f(x) = x^2 the command L = IterationList[f, 3, 2] gives you the list L = {3, 32, (32)2} = {3, 9, 27}
