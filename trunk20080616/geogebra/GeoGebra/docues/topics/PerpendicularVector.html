PerpendicularVector

PerpendicularVector[recta g]: Vector perpendicular de una recta g. Atención: Una recta con ecuación ax + by = c tiene vector perpendicular (a, b).

PerpendicularVector[vector v]: Vector perpendicular de un vector v. Atención: El vector de coordenadas (a, b) tiene uno perpendicular de (- b, a).
