Función Condicional

Puede usarse el comando Booleano Si (ver comando Si) para crear una función condicional.

Atención: Pueden usarse derivadas y integrales de tales funciones e Intersectarlas como funciones “normales”.

 

Ejemplo:

f(x) = Si[x < 3, sin(x), x^2] ofrece una función igual a:

·        sin(x) para x < 3 y

·        x2 para x ≥ 3.
