Dilata

Dilata[punto A, número f, punto S]: Dilata punto A desde el punto S usando factor f

Dilata[recta h, número f, punto S]: Dilata recta h desde punto S usando factor f

Dilata[cónica c, número f, punto S]: Dilata sección cónica c desde punto S usando factor f

Dilata[polígono poly, número f, punto S]: Dilata polígono poly desde punto S usando factor f. Atención: New vertices y segments are created too.

Dilata[image pic, número f, punto S]: Dilata image pic desde punto S usando factor f

 

Atención: Ver también modo  Dilata objeto desde punto por vector
