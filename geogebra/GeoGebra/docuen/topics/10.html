The Context Menu

The Context Menu provides a quick way to change the behavior or advanced properties of an object. Right click (Mac OS: Ctrl-click) on an object in order to open its Context Menu. For example, it allows you to change the object’s algebraic notation (e. g., polar or Cartesian coordinates, implicit or explicit equation) and to directly access features like  Rename,  Delete,  Trace On, Animation On, or  Copy to Input Bar.

 

Note: If you open the Context Menu for a point in the Graphics View, it gives you the option  Trace to Spreadsheet  (only if the Spreadsheet View is active). Once selected, this feature allows you to record the coordinates of the point in the Spreadsheet View if it is moved.

 

Note: Selecting  Properties… in the Context Menu opens the Properties Dialog, where you can change the properties of all objects used.
