Create New Tool…

Based on an existing construction you can create your own tools in GeoGebra. After preparing the construction of your tool, choose Create new tool in the Tools menu. In the appearing dialog you can specify the output and input objects of your tool and choose names for the Toolbar icon and corresponding command.

 

Note: Your tool can be used both with the mouse and as a command in the Input Bar. All tools are automatically saved in your GGB construction file.
