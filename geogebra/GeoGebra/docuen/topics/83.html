Locus

Mark a point B that depends on another point A and whose locus should be drawn. Then, click on point A.

Note: Point B has to be a point on an object (e.g. line, segment, circle).

 

Example:

·         Type f(x) = x^2 – 2 x – 1 into the input field.

·         Place a new point A on the x-axis (see mode  New point; see command Point).

·         Create point B = (x(A), f’(x(A))) that depends on point A.

·         Select mode  Locus and successively click on point B and point A.

·         Drag point A along the x-axis to see point B moving along its locus line.
