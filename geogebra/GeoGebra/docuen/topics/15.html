3.1.         General Notes

The geometry window (to the right) shows the graphical representation of points, vectors, segments, polygons, functions, straight lines, and conic sections. Whenever the mouse is moved over one of these objects a description appears and the object is highlighted.

Note: Sometimes, the geometry window will be called drawing pad.

 

There are several modes to tell GeoGebra how it should react to mouse input in the geometry window (see Modes). For example, clicking on the drawing pad can create a new point (see mode  New point), intersect objects (see mode  Intersect two objects), or create a circle (see  Circle modes).

 

Note: Double clicking an object in the algebra window opens its editing field.
