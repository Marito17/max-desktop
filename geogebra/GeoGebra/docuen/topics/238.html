6.4.         Continuity

GeoGebra allows you to turn on / off the continuity heuristic in the Options menu. The software uses a near-to-heuristic to keep moving intersection points (line-conic, conic-conic) close to their old positions and avoid jumping intersection points.

 

Note: By default this heuristic is turned off. For user defined tools (see User Defined Tools) continuity is always turned off too.
