3.1.2.        Show and Hide

Geometric objects can be drawn (show) or not (hide). Use mode  Show / hide object or the Context menu to change this state. The icon to the left of every object in the algebra window tells us about its current visibility state ( “shown” or  “hidden”).

 

Note: You can also use the mode  Check box to show and hide objects in order to show / hide one or several objects.
