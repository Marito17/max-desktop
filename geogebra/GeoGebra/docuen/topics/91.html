Text

With this mode you can create static and dynamic texts or LaTeX formulas within the geometry window.

·         Clicking on the drawing pad creates a new text field at this location.

·         Clicking on a point creates a new text field whose position is relative to this point.

 

Afterwards, a dialog appears where you may enter the text.

Note: It is also possible to use values of objects in order to create dynamic texts.

 

Input

Description

“This is a text”

simple text (static)

“Point A = ” + A

dynamic text using the value of point A

“a = ” + a + ”cm”

dynamic text using the value of segment a

 

The position of a text may be absolute on the screen or relative to the coordinate system (see Properties of the text).
