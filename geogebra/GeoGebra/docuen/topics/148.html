Point

Point[line g]: Point on line g

Point[conic c]: Point on conic section c (e.g. circle, ellipse, hyperbola)

Point[function f]: Point on function f

Point[polygon poly]: Point on polygon poly

Point[vector v]: Point on vector v

Point[point P, vector v]: Point P plus vector v
